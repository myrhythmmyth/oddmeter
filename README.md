# ODD METER — 変拍子楽曲データベース

変拍子（奇数拍子・複合拍子）の楽曲を集めて、検索・フィルタ・統計で探せる静的Webサイトです。
データはGoogleスプレッドシートを「ウェブに公開（CSV）」して読み込んでいるため、スプレッドシートに追記するだけでサイトに反映されます。

## 特徴

- 🔍 曲名・アーティスト・備考の横断検索
- 🥁 拍子タグでの絞り込み（OR / AND 切替）・分母ごとに色分け表示
- ⭐ 難易度フィルタ
- 🎚️ アーティスト一覧サイドバーから絞り込み
- 📊 拍子の分布グラフ
- ▶️ カードクリックで YouTube 埋め込み再生
- 🌗 ダーク / ライトテーマ・グラスモーフィズムUI・レスポンシブ対応

## 構成

```
index.html        … 画面構造
css/styles.css    … スタイル（グラスモーフィズム）
js/app.js         … ロジック（取得・検索・フィルタ・統計・モーダル）
js/config.js      … 接続設定（スプレッドシートCSVのURL・列マッピング）
```

ビルド不要。`index.html` をそのまま配信すれば動きます（GitHub Pages 等）。

## データ接続

`js/config.js` の `SHEET_CSV_URL` に、Googleスプレッドシートを
「ファイル → 共有 → ウェブに公開 → CSV」で公開したURLを設定します。
列の見出しは `COLUMN_MAP` で対応付けており、表記ゆれを吸収します。

## ローカルで動かす

```bash
python -m http.server 5500
# → http://localhost:5500 を開く
```
